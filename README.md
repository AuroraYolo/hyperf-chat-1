# Hyperf-Chat

## 1、简介

这是一个使用Hyperf框架的开发的IM后端应用程序。此项目是 [LumenIM-Serve](https://github.com/gzydong/LumenIM-Serve) 的重构版本。

## 3、环境要求

 - PHP >= 7.2
 - Swoole  >= 4.4
 - OpenSSL
 - JSON
 - PDO
 - Redis >= 5.0.0
 - AMQP
